	\begin{enumerate}
		\item Si $x$ est le nombre de personnes tuées sur toutes les routes, on a :
		
		$\dfrac{55}{100} \times x = \np{1911}$, d'où $x = \np{1911} \times \dfrac{100}{55} \approx \np{3474,55}$ soit \np{3475} tués à l'unité près.
