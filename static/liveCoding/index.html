Les prérequis logiciels

  • Un éditeur : VSCodium ou Visual Studio Code
  • Des extensions : Live Server, Python, Auto Import - ES6, TS, JSX, TSX de Sergey Korenuk, French Language Pack...
  • Régler la sauvegarde automatique
  • Un compte sur Github.com pour créer un fork
  • Un client Git : Github Desktop

Tester MathALEA en local

  • On ouvre le répertoire du dépôt local dans VSCodium
  • On clique sur Go Live
  • mathalea.html : les exercices
  • mathalealatex.html : le générateur LaTeX
  • cm.html : les diaporamas
  • beta.html : les exercices beta

L'arborescence

  • /exercices/ : les exercices rangés par niveau, le rangement se fera suivant les premièrs caractères 6N1, beta...
  • /modules/outils.js : ensemble de fonctions à importer à la demande dans les exercices
  • /modules/2d.js : fonctions pour générer des illustrations avec MathALEA2D
  • /modules/exercices_to_json.py : script pour mettre à jour le menu avec tous les exercices

Gabarits de littéraux


let a = randint(2,10)

Gabarits de littéraux et LaTeX


let a = randint(2,10)
let b = randint(-10,10,[-1,0,1]) // on exclut 3 valeurs
let texte = `Calcul : $${a} \\div ${b}$` // $...$ pour KaTeX 
// toutes les commandes LaTeX doivent être précédées de \\

Attention aux décimaux

import {calcul, tex_nombre, tex_nombrec} from "/modules/outils.js"
switch (liste_type_de_questions[i]) {
	case "add":
		texte = `$ ${a} + ${b} = $`;
		let somme = calcul(a+b)
		texte_corr = `$ ${a} + ${b} = ${tex_nombre(somme)} $`;
		break;
	case "mul":
		texte = `$ ${a} \\times ${b} = $`;
		texte_corr = `$ ${a} \\times ${b} = ${tex_nombrec(a * b)} $`;
		break;
