Enregistrement des scores
En cliquant sur l'un des boutons, tu peux au choix :
  • Voir la documentation. C'est mieux si tu découvres l'enregistrement des scores !
  • Enregistrer des scores dans un espace existant en renseignant un userId.

Pour enregistrer des scores dans un espace professeur existant, tu as besoin :
  • Du code prof => 3 lettres majuscules, par exmeple STI.
  • Du code classe => 2 caractères alphanumériques, les lettres sont des majuscules, par exemple 6A.
  • Le code eleve => 2 caractères alphanumériques, les lettres sont des majuscules, par exemple 1E.
