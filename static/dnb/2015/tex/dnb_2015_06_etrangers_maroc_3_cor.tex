\begin{enumerate}
\item %Quelles sont les différents codes possibles ?
A1\quad A2\quad A3\quad B1\quad B2\quad B3\quad C1\quad C2\quad C3.
\item %Aurélie compose au hasard le code A1.
	\begin{enumerate}
		\item %Quelle probabilité a-t-elle d'obtenir le bon code ?
Aurélie a un chance sur neuf : probabilité égale à $\dfrac{1}{9}$.
		\item %En tapant ce code A1, Aurélie s'est trompée à la fois de lettre et de chiffre. Elle change donc ses choix.
		
%Quelle probabilité a-t-elle de trouver le bon code à son deuxième essai ?
Si l'on supprime la lettre A et le nombre 1, il reste deux lettres et deux nombres donc $2 \times 2 = 4$ choix possibles. La probabilité  de trouver le bon code à son deuxième essai est donc égale à $\dfrac{1}{4}$.
		\item %Justifier que si lors de ce deuxième essai, Aurélie ne se trompe que de lettre,
%elle est sûre de pouvoir ouvrir la porte lors d'un troisième essai.
Comme elle n'avait plus que le choix entre deux lettres mais le bon nombre il lui suffit donc de changer de lettre pour avoir le bon code.
	\end{enumerate}
\end{enumerate}
