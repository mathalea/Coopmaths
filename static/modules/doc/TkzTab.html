Class: TkzTab

TkzTab()

Tableaux de signes, tableaux de variations, ... * tabInit est un tableau contenant sous forme de chaine les paramètres de la macro Latex \tabInit{}{} tabLines est un tableau contenant sous forme de chaine les paramètres des différentes macro \tabLine{} exemple : tabInit:[[[texte1,taille1],[texte2,taille2]...],[valeur1,valeur2,valeur3,...],[couleurs éventuelles]] tabLines:[[type,codeL1C1,codeL1C2,codeL1C3,...],[type,codeL2C1,codeL2C2,codeL2C3,...]]
